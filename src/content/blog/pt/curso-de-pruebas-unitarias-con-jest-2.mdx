Os testes unitários nos permitem testar blocos de código para ver o que funciona corretamente.  O "**Curso de Teste Unitário com Jest**" que gravei em setembro de 2018, é o primeiro dos cursos que gravei no Platzi.

Em 2019 gravei uma atualização onde focamos em adicionar testes unitários a um projeto construído em React, React Router, Redux.

Você aprenderá a configurar seu ambiente de trabalho, testar componentes, ações, redutores, gerar stapshots, criar mocks para testar uma API e ao final implementaremos os testes em um fluxo de integração contínua com travis.
