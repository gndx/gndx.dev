# Vamos falar sobre Web3

Há muito tempo venho aprendendo sobre o assunto, lendo, brincando, quebrando projetos e cheguei a uma conclusão:

Uma das maneiras de ser um desenvolvedor Web 3 é pegar suas habilidades em HTML/CSS/JS (Frameworks e Bibliotecas) e aplicá-las à Web 3.

## Mas o que é web 3?

O principal conceito a entender sobre web 3 é a forma como vamos acessar uma internet descentralizada e para isso será utilizada a tecnologia blockchain. No campo do desenvolvimento Web, #Web3 + JavaScript é uma série de bibliotecas escritas em JavaScript/TypeScript para acessar a rede blockchain independente da tecnologia utilizada (Ethereum/Solana/etc.).

Não estamos reinventando a roda, a web ainda é a web, entender como o ecossistema de desenvolvimento web 3 se move é essencial para o que vivenciaremos no futuro.
