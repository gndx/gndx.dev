**React** é uma biblioteca desenvolvida pelo Facebook que nos ajuda a construir interfaces de usuário interativas para todos os tipos de aplicativos: web, dispositivos móveis ou desktop.

Neste curso convertemos um site criado com HTML/CSS para React, onde entendemos como criar passo a passo um projeto React onde configuramos nosso ambiente de trabalho, webpack, babel, ambiente de desenvolvimento local e depois criamos todos os componentes necessários para criar o Platzi Video.
