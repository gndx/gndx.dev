Quando estamos começando como "**Desenvolvedores Web com JavaScript**" nos deparamos com termos, nomenclaturas e abreviações que podem nos confundir, entre os quais encontramos: ES5, ES6, ES2015, ECMAScript 2017 e assim por diante um grande número deles.

**O que é ECMAScript?**

ECMAScript é o padrão no qual o JavaScript se baseia e você o encontrará em documentos, livros ou recursos na internet com a abreviatura "**ES**" ou simplesmente **ECMAScript**.

O JavaScript não usa apenas o padrão proposto pelo ECMAScript, mas também é implementado pelo ActionScript, linguagem usada pelo Flash, e pelo JScript, uma adaptação do JavaScript para o Internet Explorer.
