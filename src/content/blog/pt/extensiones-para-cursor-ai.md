O Cursor AI é completamente transformado com as extensões certas. Desde impulsionar seu fluxo de trabalho até evitar erros comuns e ganhar velocidade, esta seleção cobre o essencial para desenvolvedores modernos que procuram um ambiente ágil, organizado e visualmente atraente.

### 📦 Lista de extensões para Cursor AI 2025

1. **Estrela**
   Suporte total para a estrutura Astro, incluindo destaque de sintaxe, diagnóstico e preenchimento automático.
   🔗 [Astro - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=astro-build.astro-vscode)

2. ** Corredor de código **
   Execute trechos de código em vários idiomas diretamente do editor.
   🔗 [Code Runner - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=formulahendry.code-runner)

3. **DotENV**
   Fornece realce de sintaxe e outros recursos para arquivos `.env`.
   🔗 [DotENV - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=mikestead.dotenv)

4. **Erro de lente**
   Melhora a visibilidade de erros e avisos diretamente no editor.
   🔗 [Erro de lente - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=usernamehw.errorlens)

5. **Snippets ES7 React/Redux/GraphQL/React-Native**
   Snippets para acelerar o desenvolvimento com React, Redux e GraphQL.
   🔗 [Snippets ES7 React/Redux/GraphQL/React-Native - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=dsznajder.es7-react-js-snippets)

6. **ESLint**
   Integração ESLint para manter código limpo e consistente.
   🔗 [ESLint - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)

7. **GitLens — Git sobrecarregado**
   Ferramenta poderosa para visualizar e entender o histórico do seu código com Git.
   🔗 [GitLens - Git sobrecarregado - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens)

8. **Visualização da imagem**
   Visualize imagens diretamente no editor.
   🔗 [Visualização da imagem - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=kisstkondoros.vscode-gutter-preview)

9. **Custo de importação**
   Mostra o tamanho dos pacotes importados em tempo real.
   🔗 [Custo de importação - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=wix.vscode-import-cost)

10. **indent-arco-íris**
    Recuos de cores para melhorar a legibilidade do código.
    🔗 [indent-rainbow - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=oderwat.indent-rainbow)

11. **Tema do ícone de material**
    Aplica um tema de ícones baseado em Material Design para melhor visualização de arquivos.
    🔗 [Tema do ícone de material - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=PKief.material-icon-theme)

12. **MDX**
    Suporte para arquivos MDX, combinando Markdown com JSX.
    🔗 [MDX - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=silvenon.mdx)

13. **Próximos trechos de js**
    Snippets para acelerar o desenvolvimento com Next.js.
    🔗 [Trechos de Nextjs - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=andys8.vscode-nextjs-snippets)

14. **npm Intellisense**
    Preenchimento automático para módulos npm em seus arquivos de código.
    🔗 [npm Intellisense - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=christian-kohler.npm-intellisense)

15. **Mais bonito - Formatador de código**
    Formate automaticamente seu código para manter um estilo consistente.
    🔗 [Mais bonito - Formatador de código - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)

16. **CSV arco-íris**
    Colorir e facilitar a edição de arquivos CSV.
    🔗 [Rainbow CSV - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=mechatroner.rainbow-csv)

17. **Tailwind CSS IntelliSense**
    Preenchimento automático e sugestões para classes CSS do Tailwind.
    🔗 [Tailwind CSS IntelliSense - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss)

18. **Espaços finais**
    Destaca e remove espaços em branco desnecessários no final das linhas.
    🔗 [Espaços finais - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=shardulm94.trailing-spaces)

19. **YAML**
    Suporte aprimorado para arquivos YAML, úteis em configurações e Kubernetes.
    🔗 [YAML - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=redhat.vscode-yaml)

Essas extensões estão disponíveis no Visual Studio Code Marketplace oficial e são compatíveis com Cursor AI, já que este editor é baseado na plataforma VS Code.
