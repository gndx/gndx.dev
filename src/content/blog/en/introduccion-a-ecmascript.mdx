When we are getting started as "**Web Developers with JavaScript**" we come across terms, nomenclatures and abbreviations that can confuse us, among which we find: ES5, ES6, ES2015, ECMAScript 2017 and so on a large number of these.

**What is ECMAScript?**

ECMAScript is the standard on which JavaScript is based and you will find it in documents, books or resources on the internet with the abbreviation "**ES**" or simply **ECMAScript**.

Not only does JavaScript use the standard proposed by ECMAScript, it is also implemented by ActionScript, the language used by Flash, and JScript, an adaptation of JavaScript for Internet Explorer.
